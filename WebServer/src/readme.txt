to run this project successfully,you need to do below steps:
	1.change database conf info in hibernate.cfg.xml.
	2.run junit test method 'testCreate'(com.pitaya.avatar.test.Test) to create table in database.
	3.run method 'testAddSchool'(com.pitaya.avatar.test.Test) to insert record in database.
	4.deploy project in J2EE server(tomcat,jboss etc).
	5.run j2EE server.
	6.access path in browser:http://localhost:port/pitaya-avatar
	7.input '三牌楼小学' and click '按学校查询'
